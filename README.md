| Flatpak Apps | External Apps |
| -------------|---------------|
| Slack        | Google Chrome |
| GitHub Desktop | node.js (via nvm) |
| Zoom         | Git           |
| VSCode       | 

# Instruções de instalação:

```
sudo apt install curl
